## What is dd-trace?

`dd-trace` is the official Datadog APM (Application Performance Monitoring) client library for Node.js. It automatically instruments your application to capture distributed traces, which are then sent to the Datadog Agent and on to the Datadog platform for visualization, alerting, and analysis.

```bash
npm install dd-trace
```

```javascript
// Must be imported BEFORE any other modules
const tracer = require('dd-trace').init()
```

## How it works

```
Your Node.js App  →  dd-trace  →  Datadog Agent  →  Datadog Platform
```

`dd-trace` hooks into Node.js module loading to automatically wrap supported libraries. When your application handles a request, dd-trace creates a **trace** — a tree of **spans** representing each operation (HTTP call, database query, etc.). These traces are batched and sent to the local Datadog Agent over HTTP, which forwards them to the Datadog backend.
